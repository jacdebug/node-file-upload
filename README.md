# Zero dependency node file upload
Pure nodejs file upload with zero dependencies. 

## Usage
#### Clone 
```
git clone https://github.com/jacdebug/node-file-upload.git
cd node-file-upload
```

#### Run server
```
npm start
```
or just
```
node index.js
````

#### Test
```
curl --upload-file ./<file-to-upload> http://127.0.0.1:3000/
```
